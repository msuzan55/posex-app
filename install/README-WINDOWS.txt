PosEx Windows app
=================

1. Unzip this folder to a normal path (example: C:\PosEx\).
   Do NOT run the .exe from inside the ZIP / RAR / Downloads preview.

2. Prefer: double-click  Launch PosEx.cmd
   Or:     posex_app.exe

3. Requirements
   - Windows 10/11 (64-bit)
   - Microsoft Edge WebView2 Runtime
     https://go.microsoft.com/fwlink/p/?LinkId=2124703
   - Visual C++ 2015–2022 Redistributable (x64)
     Run vc_redist.x64.exe from this folder if the app will not open.

4. If the window opens then closes / blank screen
   - Close PosEx
   - Delete folder:  %APPDATA%\posex_app\webview2
   - Open Launch PosEx.cmd again

5. The app loads: https://posex.lk/test/
